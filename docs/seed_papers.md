# Seed Papers

These are the starting papers for the initial reading and reproduction queue. Dates below are the arXiv posting dates.

## Memory-State
- `Transformers are SSMs: Generalized Models and Efficient Algorithms Through Structured State Space Duality` (May 31, 2024): Mamba-2 and the attention/SSM bridge. https://arxiv.org/abs/2405.21060
- `Learning to (Learn at Test Time): RNNs with Expressive Hidden States` (July 5, 2024): test-time learning layers as attention replacements. https://arxiv.org/abs/2407.04620
- `Titans: Learning to Memorize at Test Time` (December 31, 2024): explicit neural long-term memory. https://arxiv.org/abs/2501.00663
- `ATLAS: Learning to Optimally Memorize the Context at Test Time` (May 29, 2025): higher-capacity memory management and long-context results. https://arxiv.org/abs/2505.23735
- `Compressive Transformers for Long-Range Sequence Modelling` (November 13, 2019): useful precursor for memory compression. https://arxiv.org/abs/1911.05507

## Adaptive Inference
- `Adaptive Computation Time for Recurrent Neural Networks` (March 29, 2016): learned halting and variable compute. https://arxiv.org/abs/1603.08983
- `Universal Transformers` (July 10, 2018): recurrent-depth transformer with dynamic halting. https://arxiv.org/abs/1807.03819
- `Training Large Language Models to Reason in a Continuous Latent Space` (December 9, 2024): COCONUT and latent-space reasoning. https://arxiv.org/abs/2412.06769
- `Large Language Diffusion Models` (February 14, 2025): LLaDA as a serious diffusion LM baseline. https://arxiv.org/abs/2502.09992
- `Mercury: Ultra-Fast Language Models Based on Diffusion` (June 17, 2025): practical speed-quality evidence for diffusion LMs. https://arxiv.org/abs/2506.17298

## Hierarchical Programs
- `Neural Turing Machines` (October 20, 2014): explicit differentiable memory and algorithmic behavior. https://arxiv.org/abs/1410.5401
- `Recurrent Independent Mechanisms` (September 24, 2019): modular recurrent computation with sparse communication. https://arxiv.org/abs/1909.10893
- `Coordination Among Neural Modules Through a Shared Global Workspace` (March 1, 2021): structured communication between specialized modules. https://arxiv.org/abs/2103.01197
- `Byte Latent Transformer: Patches Scale Better Than Tokens` (December 13, 2024): learned dynamic patches and tokenizer-free scaling. https://arxiv.org/abs/2412.09871

## Cross-Cutting Theory
- `White-Box Transformers via Sparse Rate Reduction: Compression Is All There Is?` (November 22, 2023): interpretable transformer-like blocks from an explicit objective. https://arxiv.org/abs/2311.13110
